# Day 3 - Data Preprocessing

## 🎯 Objective
Learn how to inspect and prepare data before building a machine learning model.

---

## 📂 Dataset
Student Performance Dataset

---

## 📚 Topics Covered

- Checking missing values
- Checking duplicate records
- Identifying numerical and categorical columns
- Counting unique values
- Understanding categorical features
- Data exploration using Pandas

---

## 🛠 Libraries Used

- Pandas
- NumPy

---

## 📋 Tasks Performed

- Loaded the dataset
- Checked dataset information
- Verified missing values
- Verified duplicate records
- Identified numerical and categorical features
- Counted unique values in each column
- Analyzed internet access and higher education columns

---

## 📌 Key Findings

- The dataset contains both numerical and categorical features.
- No missing values were found.
- No duplicate records were found.
- Most students have internet access.
- Most students want to pursue higher education.
- The dataset is clean and ready for feature engineering.

- Data Cleaning
- Data Inspection
