# Day 4 - Feature Engineering

## 🎯 Objective

Learn how to transform categorical data into numerical data so that machine learning algorithms can process it effectively.

---

## 📂 Dataset

Student Performance Dataset

---

## 📚 Topics Covered

- Feature Engineering
- Identifying Numerical and Categorical Features
- Label Encoding
- One-Hot Encoding
- Saving the Processed Dataset

---

## 🛠 Libraries Used

- Pandas
- NumPy
- Scikit-learn (LabelEncoder)

---

## 📋 Tasks Performed

- Loaded the Student Performance dataset
- Identified numerical and categorical columns
- Applied Label Encoding to binary categorical features
- Applied One-Hot Encoding to multi-category features
- Verified that all columns are numerical
- Saved the processed dataset

## 📊 Key Learnings

- Machine learning algorithms cannot work directly with text values.
- Label Encoding is suitable for binary categorical variables.
- One-Hot Encoding is suitable for categorical variables with more than two categories.
- Feature Engineering is an important preprocessing step before training a machine learning model.
